# 주차 관제 어플리케이션 프로젝트 (클라우드처럼 사용)
flutter dart code lib

main -> 초기 로그인 화면
temp -> 비밀번호 변경 화면, 지도 화면, 앱 종료 화면
init -> 챠랑 번호, 기사 연락처 등 초기화 화면
send -> 킨텍스 갈 때 화면 (터미널 경유의 경우)
send1 -> 킨텍스 갈 때 화면 (주차장 정차의 경우)
receive -> 집 올 때 화면 (터미널 경유의 경우)
receive1 -> 집 올 때 화면 (주차장 정차의 경우)
change -> send와 receive의 비밀번호 변경 화면
